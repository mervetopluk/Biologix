# BioLogix - DNA Mutation and Evolutionary Distance Calculator

## Project Description

BioLogix is a Python-based DNA Mutation and Evolutionary Distance Calculator.
This project compares two DNA sequences, calculates the number of mutations between them using Hamming Distance, and determines the evolutionary distance as a percentage. The program generates a detailed mutation report including mutation count, mutation percentage, evolutionary distance, and risk classification.
The project serves as an educational simulation that combines mathematics, programming, and bioinformatics concepts. It demonstrates how computational approaches can be used to analyze biological data and support data-driven decision-making.



## Project Purpose

The purpose of this project is to develop a Python program that compares two DNA sequences and calculates the number of mutations between them. By using mathematical sequence comparison logic, the program measures the evolutionary distance as a percentage.
This project is designed as an educational simulation that connects mathematics, programming, and bioinformatics. It may also provide a simple example of how data-driven thinking can support healthcare logistics and planning.



## Features

- DNA sequence validation
- Hamming Distance calculation
- Mutation percentage analysis
- Evolutionary distance estimation
- Risk level classification
- Automatic mutation report generation
- Save report as a TXT file



## How to Run

1. Run the "biologix.py" file.
2. Enter the original DNA sample name.
3. Enter the original DNA sequence.
4. Enter the variant DNA sample name.
5. Enter the variant DNA sequence.
6. View the generated mutation report.
7. Optionally save the report as a text file.
